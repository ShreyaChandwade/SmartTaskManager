#  Smart Task Manager App

Smart Task Manager is an Android productivity application that helps users manage daily tasks efficiently using reminders, priority levels and a simple user interface.

---

## Features

- User Login & Signup authentication
- Add / Update / Delete Tasks
- Task Priority & Reminder system
- Calculator Module
- Contacts Module
- Accelerometer Sensor Feature (Shake Detection)
- SQLite Database Integration
- Material UI Design

---

## Tech Stack

- Kotlin
- XML (Android UI)
- SQLite Database
- Android Sensors (Accelerometer)
- Android Studio

---

## Project Structure

app/
 ├── java → Kotlin source code  
 ├── res → Layouts, Colors, Strings  
 ├── manifests → AndroidManifest.xml  

---

## How to Run This Project

1. Download or clone the repository
2. Open project in Android Studio
3. Sync Gradle
4. Run on Emulator or Physical Device
